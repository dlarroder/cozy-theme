# Changelog

## 0.2.0

- Switched to warm charcoal background palette (no more purple tint)
- Improved object key vs property access distinction with semantic highlighting
- Softened function color to soft sky (`#82b8cc`)
- Updated string color to dusty coral (`#c49a8a`)
- Added dedicated colors for React component tags (`#d480b8`)
- HTML attribute names now use soft sky with italic styling
- Fixed git gutter and terminal ANSI green colors

## 0.1.0

- Initial release
- Dark theme with warm, cozy color palette
- Semantic highlighting support
- Full terminal color support
- Syntax highlighting for popular languages (JS/TS, Python, Rust, Go, HTML/CSS, JSON, Markdown, and more)
